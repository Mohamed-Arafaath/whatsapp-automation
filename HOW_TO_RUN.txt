HOW TO RUN THIS APP (Windows & Mac)
===================================

You can run this WhatsApp Automation app on any computer!

PREREQUISITES
-------------
1. Install Python: https://www.python.org/downloads/
   (Make sure to check "Add Python to PATH" during installation on Windows)

2. Install Google Chrome: https://www.google.com/chrome/

HOW TO START
------------

OPTION 1: Mac
1. Open Terminal.
2. Navigate to this folder.
3. First time only: Run ./SETUP_MAC.sh
   (This installs Python, Chrome, and everything else)
4. To start app: Run ./RUN_MAC.sh

OPTION 2: Windows
1. First time only: Double-click "SETUP_WINDOWS.bat"
   (This installs Python, Chrome, and everything else)
2. To start app: Double-click "RUN_WINDOWS.bat"

OPTION 3: Manual (Any OS)
1. Open Terminal/Command Prompt.
2. Install dependencies:
   pip install -r requirements.txt
3. Start the app:
   python app.py

Once running, open your browser and go to:
http://localhost:5001

ENJOY!
