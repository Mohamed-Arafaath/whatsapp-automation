HOW TO RUN THIS APP (Windows & Mac)
===================================

You can run this WhatsApp Automation app on any computer!

PREREQUISITES
-------------
1. Install Python: https://www.python.org/downloads/
   (Make sure to check "Add Python to PATH" during installation on Windows)

2. Install Google Chrome: https://www.google.com/chrome/

HOW TO START
------------

OPTION 1: Mac
1. Download the "Local/Mac" folder.
2. Open Terminal and go to that folder.
3. First time only: Run ./SETUP_MAC.sh
4. To start app: Run ./RUN_MAC.sh

OPTION 2: Windows
1. Download the "Local/Windows" folder.
2. First time only: Double-click "SETUP_WINDOWS.bat"
3. To start app: Double-click "RUN_WINDOWS.bat"

OPTION 3: Cloud (Render)
1. Use the "Cloud" folder for deployment.
2. Set Dockerfile Path to "Cloud/Dockerfile".

Once running, open your browser and go to:
http://localhost:5001

ENJOY!
